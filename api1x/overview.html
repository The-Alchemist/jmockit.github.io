JMockit is a Java toolkit for automated developer testing. The figure below provides an overview of the available APIs. Clicking on individual items will navigate to the corresponding API documentation or Tutorial pages.


Figure 1. Overview of JMockit APIs

For a quick introduction to JMockit, go to the Getting Started page. For the motivations that led to the creation of the toolkit, see the About JMockit page.

The JMockit Tutorial contains an extensive description of the APIs and the code coverage tool, with plenty of example code.

Bug reports, questions, feature/enhancement requests and other issues should be posted on the Issue Tracker. Source code contributions can be submitted as regular GitHub pull requests from a forked repository.
